# Docker ELK

Run the latest version of the ELK (Elasticsearch, Logstash, Kibana) with Docker and Docker Compose.

Based on the official Docker images:

* [docker pull elasticsearch](https://hub.docker.com/_/elasticsearch/)
* [docker pull logstash](https://hub.docker.com/r/_/logstash/)
* [docker pull kibana](https://hub.docker.com/_/kibana/)
